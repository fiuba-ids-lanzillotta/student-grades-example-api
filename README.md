# 2025C1-EJ-Integral
Repo para el ejercicio que integra tanto los conceptos vistos de frontend como de backend.

Para correr el back y el front por separado, hay que correrlos como modulos:

`python -m app_backend.app`

y 

`python -m app_frontend.app`

Para correrlo todo junto (por si se quiere hacer el deploy como una sola app en, por ejemplo, pythonanywhere), hay que corre:

`python main_app.py`
